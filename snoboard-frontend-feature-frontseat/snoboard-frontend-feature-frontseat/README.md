# SnoBoard - Campaign Intelligence Platform

Professional influencer campaign management and analytics platform.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Recharts
- Framer Motion
- Supabase

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```
